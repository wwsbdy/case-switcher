# case-switcher

<p>Use <b>Shift + Alt + ,</b> and <b>Shift + Alt + .</b> to quickly cycle selected text through different naming styles.</p>
<ul>
  <li>Supported naming styles:</li>
  <ul>
    <li>camelCase</li>
    <li>CamelCase</li>
    <li>camel_case</li>
    <li>CAMEL_CASE</li>
    <li>camel-case</li>
    <li>CAMEL-CASE</li>
    <li>camel case</li>
    <li>Camel Case</li>
    <li>CAMEL CASE</li>
  </ul>
  <li>Settings page allows you to enable/disable styles and adjust their order of cycling.</li>
</ul>
